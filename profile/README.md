## Specification

The Flow-Based Programming specification lives at [**github.com/flow-based-programming/spec**](https://github.com/flow-based-programming/spec).

## Mission

We build open source tools for flow-based programming — giving developers the primitives, runtimes, and tooling they need to design software as networks of independent, message-passing components.
